Отчет реализации проекта "Веб-приложение для предсказания качества вина" с использованием модели машинного обучения и датасета WineQuality.

1. Введение:
Цель проекта состояла в создании веб-приложения, которое позволяет пользователям предсказывать качество вина на основе различных характеристик. 
Для этого использовался датасет WineQuality, содержащий информацию о химических характеристиках вина и его оценке качества.

2. Анализ данных:
Для начала проекта был проведен анализ датасета WineQuality. Были изучены статистические характеристики признаков, распределение данных, 
наличие пропущенных значений и выбросов. Проведенные анализ позволил лучше понять характеристики данных и выбрать соответствующие методы предобработки.

3. Предобработка данных:
В этом шаге проекта была проведена предобработка данных для подготовки их к обучению модели. Были выполнены следующие действия:
   - Обработка пропущенных значений: пропущенные значения в датасете были заполнены с помощью метода K-средних.
   - Преобразование категориальных признаков: категориальный признак "тип вина" был закодирован числовыми значениями с использованием метода Label Encoding.
   - Удаление выбросов: были удаления значения, которые лежат за пределами квантилей 0.25 и 0.75. После удаления было потеряно только 3.51% данных

4. Обучение модели:
Для предсказания качества вина была выбрана модель машинного обучения - ExtraTreesRegressor. 
Были подобраны гиперпараметры модели с помощью библиотеки Optuna. 
Модель была обучена на обработанных данных с использованием метода кросс-валидации для получения более надежных результатов. 
После обучения модели были оценены ее метрики качества, такие как среднеквадратическая ошибка (MSE).

5. Создание веб-приложения:
С использованием библиотеки Streamlit было разработано веб-приложение, которое позволяет пользователям взаимодействовать с моделью предсказания качества вина.
 Веб-приложение имеет простой и интуитивно понятный интерфейс, где пользователь может ввести значения химических характеристик вина и получить предсказание его качества. 

6. Заключение:
В результате реализации проекта было создано веб-приложение, которое позволяет пользователям предсказывать качество вина на основе его химических характеристик. 
Использование модели машинного обучения ExtraTreesRegressor позволяет получать достаточно точные предсказания. 
Проект может быть полезен для любителей вина, сомелье, виноделов и других заинтересованных лиц, 
которые хотят получить оценку качества вина на основе его химических характеристик.
